# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos, or single-package repos to help you version and publish your code.

## How to add a changeset

1. Run `npx changeset` in the root of the repository
2. Follow the prompts to describe your changes
3. Commit the generated changeset file

## How to release

1. Run `npx changeset version` to bump versions based on changesets
2. Run `npx changeset publish` to publish to npm
3. Or use the GitHub Actions workflow for automated releases

For more information, see [the changesets documentation](https://github.com/changesets/changesets).
